Cómo navegar el curso

Elige tu entrada

Esta página separa el recorrido operativo del curso en dos rutas: estudiante y profesor. El objetivo es que cada persona sepa dónde empezar, qué abrir durante la semana y dónde buscar evidencias, entregas o materiales de clase.

Estudiante

Qué hago esta semana

  1. Abre la ruta semanal.
  2. Ubica la semana actual.
  3. Lee el capítulo indicado en el libro completo.
  4. Usa las diapositivas y el notebook de clase.
  5. Entrega solo el archivo marcado como laboratorio, parcial o hito de proyecto.

Profesor

Qué preparo y qué reviso

  1. Consulta la ruta semanal para ubicar el bloque de clase.
  2. Abre el panel docente para preparar antes, durante y después de clase.
  3. Usa materiales para validar enlaces públicos.
  4. Usa las rúbricas para revisar solo interpretación, decisiones técnicas y casos marcados.
  5. Contrasta fechas y entregas con el calendario.

Regla de navegación

Portal para operar, libro para estudiar. Si necesitas saber qué entregar o qué abrir en clase, usa el portal. Si necesitas entender el tema, demostrar un resultado o practicar ejercicios, usa el libro.

Señales importantes

Antes de clase

Lectura del libro y, cuando aplique, preparación del entorno.

Durante clase

Presentación y notebook de live coding.

Para entregar

Notebook de laboratorio, parcial autograded o hito del proyecto.

Para estudiar

Capítulos, ejercicios, respuestas seleccionadas, glosario y formularios.
