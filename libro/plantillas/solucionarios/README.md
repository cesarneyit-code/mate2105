# Paquete de respuestas y solucionarios

Este paquete contiene plantillas públicas para controlar qué respuestas pueden
publicarse y qué recursos deben permanecer restringidos.

## Archivos

| Archivo | Uso |
|---|---|
| `manual_estudiante_respuestas.md` | preparar una clave pública parcial para estudiantes |
| `manual_instructor_restringido.md` | estructurar un manual restringido sin publicarlo |
| `registro_liberacion_respuestas.md` | documentar si una respuesta puede hacerse pública |

## Regla editorial

Las plantillas pueden publicarse. Las respuestas completas de evaluaciones,
pruebas privadas, tolerancias y casos límite no deben publicarse.
