# Checklist de revisión de impresión

Usar esta lista antes de distribuir el PDF como versión estable o antes de pedir
copias impresas.

- Libro:
- Edición:
- Fecha de render:

## Muestra revisada

- [ ] Portada
- [ ] Índice
- [ ] Información editorial
- [ ] Capítulo con derivaciones matemáticas
- [ ] Capítulo con bloques de código
- [ ] Capítulo con figuras
- [ ] Banco de ejercicios
- [ ] Apéndice de respuestas o fórmulas
- [ ] Página final

## Criterios visuales

- [ ] No hay títulos aislados al final de página.
- [ ] Las tablas no se salen del área imprimible.
- [ ] Las fórmulas largas no quedan cortadas.
- [ ] Los bloques de código son legibles.
- [ ] Las figuras conservan caption y ubicación razonable.
- [ ] Las listas no quedan cortadas de forma confusa.
- [ ] El contraste funciona en color.
- [ ] El contraste funciona en escala de grises.
- [ ] Los enlaces visibles no son indispensables para entender el texto impreso.

## Criterios técnicos

- [ ] Tamaño carta confirmado.
- [ ] Metadatos de título y autor confirmados.
- [ ] Texto extraíble confirmado.
- [ ] Sin JavaScript confirmado.
- [ ] Sin encriptación confirmado.
- [ ] Estado de PDF etiquetado declarado.
- [ ] Manifiesto de publicación actualizado.

## Resultado

- [ ] Aprobado para uso del curso
- [ ] Aprobado para impresión controlada
- [ ] Requiere correcciones antes de impresión
