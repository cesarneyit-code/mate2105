# Paquete de errata y control de cambios

Este directorio contiene plantillas para reportar, priorizar y cerrar errata del
libro MATE-2105.

## Archivos

| Archivo | Uso |
|---|---|
| `reporte_errata.md` | formulario para reportar un problema del libro público |
| `registro_errata_publica.md` | tabla pública de cambios cerrados o pendientes |
| `cierre_correccion.md` | checklist para cerrar una corrección con evidencia |
| `clasificacion_severidad.md` | criterios para clasificar severidad e impacto |

## Regla editorial

La errata pública puede documentar problemas de texto, fórmulas, código público,
figuras, tablas y enlaces. No debe revelar soluciones oficiales, pruebas ocultas,
bancos de parcial ni detalles internos del autograder.
