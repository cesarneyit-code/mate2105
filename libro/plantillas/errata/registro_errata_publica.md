# Registro público de errata

Este registro resume correcciones visibles para lectores del libro. Las
correcciones que revelen evaluación privada deben registrarse solo en el espacio
docente restringido.

| ID | Fecha reporte | Sección | Tipo | Severidad | Estado | Fecha cierre |
|---|---|---|---|---|---|---|
| E-000 | 2026-07-07 | front matter | consolidación editorial | baja | cerrado | 2026-07-07 |

## Estados

| Estado | Significado |
|---|---|
| recibido | el reporte fue recibido pero no clasificado |
| en revisión | se está verificando el problema |
| aceptado | se confirmó que requiere corrección |
| cerrado | se aplicó corrección y hay evidencia |
| rechazado | no se acepta como errata, con justificación |
| privado | se movió a seguimiento docente restringido |

## Reglas de publicación

- No publicar datos personales del reportante sin autorización.
- No publicar soluciones oficiales ni detalles de autograders.
- Incluir solo el resumen necesario para que el lector entienda el cambio.
- Registrar evidencia de cierre en la plantilla correspondiente.
