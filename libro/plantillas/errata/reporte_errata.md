# Reporte de errata

Use este formulario para reportar un problema en el libro público.

## Identificación

```text
Fecha:
Reportado por:
Contacto opcional:
Versión o URL consultada:
Capítulo o página:
Sección:
```

## Tipo de problema

Marque una o varias opciones:

- [ ] Fórmula o derivación.
- [ ] Resultado numérico.
- [ ] Código público.
- [ ] Figura o texto alternativo.
- [ ] Tabla.
- [ ] Enlace roto.
- [ ] Redacción o ambigüedad.
- [ ] Accesibilidad.
- [ ] PDF.
- [ ] Otro:

## Descripción

```text
Qué dice actualmente:
Cuál es el problema:
Por qué afecta el aprendizaje o la publicación:
```

## Evidencia

Incluya una de estas evidencias si aplica:

- cálculo manual;
- error de ejecución;
- captura;
- enlace roto;
- comparación con notebook;
- página del PDF;
- mensaje de auditoría.

## Corrección sugerida

```text
Corrección propuesta:
Archivos posiblemente afectados:
Impacto esperado:
```

## Control de material privado

- [ ] El reporte no incluye soluciones oficiales.
- [ ] El reporte no incluye pruebas ocultas.
- [ ] El reporte no revela lógica de autograding.
