# Paquete de revisión externa

Este directorio contiene plantillas para solicitar, registrar y cerrar una
revisión externa del libro MATE-2105. Las plantillas no sustituyen el juicio del
revisor; convierten ese juicio en evidencia trazable.

## Archivos

| Archivo | Uso |
|---|---|
| `carta_solicitud_revision.md` | mensaje base para invitar a un revisor |
| `checklist_paquete_revisor.md` | lista de materiales que debe recibir el revisor |
| `formulario_dictamen_revision.md` | formato de dictamen externo |
| `rubrica_revision_externa.md` | criterios de revisión matemática, pedagógica, computacional y de accesibilidad |
| `registro_correcciones_revision.md` | seguimiento de correcciones obligatorias y recomendadas |

## Regla editorial

No se deben registrar nombres, afiliaciones ni aprobaciones si no provienen de
una revisión real. Una edición puede estar lista para el curso sin estar lista
como edición publicable final.
