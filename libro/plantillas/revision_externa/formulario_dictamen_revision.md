# Formulario de dictamen de revisión externa

## Identificación

```text
Revisor:
Afiliación:
Correo o identificador institucional:
Rol de revisión:
Fecha:
Versión revisada:
URL o ruta del libro revisado:
Manifest revisado:
```

## Alcance

Indique exactamente qué revisó:

- partes o capítulos;
- apéndices;
- figuras;
- PDF;
- notebooks o recursos externos, si aplica;
- elementos explícitamente no revisados.

## Dictamen

Marque una opción:

- [ ] Aprobar para uso del curso.
- [ ] Aprobar para publicación web controlada con correcciones menores.
- [ ] Revisar de nuevo antes de publicar.
- [ ] No aprobar en el estado actual.

## Comentarios por dimensión

### Matemática

Consistencia de notación, derivaciones, fórmulas, dimensiones, ejemplos y
ejercicios cuantitativos.

### Pedagógica

Secuencia, objetivos, carga de lectura, ejercicios, retroalimentación y conexión
con evaluación.

### Computacional

Código, reproducibilidad, datasets, semillas, dependencias, métricas y riesgo de
leakage.

### Accesibilidad y publicación

Navegación, texto alternativo, tablas, contraste, PDF, lector de pantalla,
manifest y separación público/privado.
