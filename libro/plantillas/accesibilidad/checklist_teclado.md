## Criterios

- [ ] Se puede llegar al contenido principal sin usar mouse.
- [ ] El orden de foco sigue la lectura visual.
- [ ] Los enlaces tienen texto comprensible fuera de contexto.
- [ ] La búsqueda se puede abrir, usar y cerrar con teclado.
- [ ] Los bloques de código reciben foco cuando necesitan desplazamiento.
- [ ] No hay trampas de teclado.
- [ ] El foco visible es claro.
- [ ] La navegación lateral no impide leer el contenido.
- [ ] Las tablas pueden desplazarse sin perder contexto esencial.
