## Criterios

- [ ] El título de cada página se anuncia correctamente.
- [ ] Hay un encabezado principal claro.
- [ ] La jerarquía de encabezados permite navegar la página.
- [ ] Las figuras tienen alt text o explicación cercana suficiente.
- [ ] Las tablas anuncian encabezados.
- [ ] Los enlaces son comprensibles.
- [ ] Las fórmulas centrales tienen explicación textual cercana.
- [ ] Los bloques de código no interrumpen la navegación.
- [ ] Las descargas de PDF y CSV se identifican claramente.
