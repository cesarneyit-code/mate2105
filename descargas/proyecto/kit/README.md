# Kit del proyecto MATE-2105

Este kit contiene la estructura mínima para los hitos y la entrega final del proyecto. La especificación vigente de fechas, puntajes y nombres de archivo está en:

<https://cesarneyit-code.github.io/mate2105/proyecto.html>

## Cómo empezar

1. Duplica esta carpeta y cámbiale el nombre a `grupo##_proyecto`.
2. Completa `data/README.md` antes de modelar.
3. Usa `plantillas/plantilla_hitos.md` para las Semanas 2, 3 y 4.
4. Adapta `src/pipeline.py` al tipo de problema.
5. Ejecuta `ejemplo/ejemplo_minimo.py` para comprobar el entorno.
6. Conserva resultados canónicos en `artifacts/metrics.json` y datos de ejecución en `artifacts/manifest.json`.
7. Antes de cada entrega, descomprime una copia del `.zip` y sigue el `README.md` como lo haría otra persona.

## Estructura

```text
mate2105_proyecto/
  README.md
  requirements.txt
  data/README.md
  notebooks/README.md
  src/pipeline.py
  reports/README.md
  artifacts/README.md
  plantillas/
  ejemplo/
```

No incluyas credenciales, datos identificables, `.git`, cachés ni rutas locales.
