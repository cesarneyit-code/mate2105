# Artefactos

Guarda aquí salidas pequeñas que permitan verificar el resultado principal:

- `metrics.json`: baseline, modelo final, partición y métricas;
- `manifest.json`: entorno, semilla, datos, comando y artefactos;
- figuras o tablas finales, si no se regeneran durante la ejecución.

No uses esta carpeta para resultados exploratorios sin relación con el reporte.
