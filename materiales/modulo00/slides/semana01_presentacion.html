El objeto aleatorio viene antes del archivo

\[ (X,Y):(\Omega,\mathcal F,\mathbb P)\longrightarrow \mathcal X\times\mathcal Y \]

  • \(\Omega\) reúne resultados posibles del proceso que genera una observación;
  • \(\mathcal F\) especifica los eventos a los que asignamos probabilidad;
  • \(\mathbb P\) describe su distribución;
  • \((X,Y)\) transforma un resultado en variables que podemos analizar.

La tabla contiene realizaciones de estos objetos, no los objetos aleatorios mismos.

Variable aleatoria y realización

Antes de observar

\(Y\) es una variable aleatoria. Su valor depende de cuál observación produzca el proceso.

\[Y:\Omega\to\mathbb R\]

Después de observar

\(y_i=10.751\) es un valor realizado para una subzona particular del archivo.

No tiene distribución una vez fijado.

¿Cuál de las dos expresiones puede tener esperanza y varianza: \(Y\) o \(y_i\)?

Una muestra es también un objeto aleatorio

\[ \mathsf D_m=((X_1,Y_1),\ldots,(X_m,Y_m)) \overset{\mathrm{iid}}{\sim}P^m \]

La notación iid afirma dos cosas que deben discutirse, no suponerse por costumbre:

  1. cada par tiene la misma distribución \(P\);
  2. conocer una observación no cambia la distribución de las demás.

Registros temporales, grupos familiares o mediciones repetidas pueden violar independencia.

Cuatro niveles que no mezclaremos

Nivel Objeto Ejemplo
población \(P\), \(\mathbb E_P[Y]\), \(R_P(f)\) distribución conceptual de observaciones futuras
muestra aleatoria \(\mathsf D_m\) el conjunto antes de ser observado
estadístico o estimador \(\bar Y_m\), \(\widehat R_m(f)\) regla que depende de la muestra
valor calculado \(\bar y_m\), \(\widehat r_m(f)\) número producido por este archivo

Confundir el tercer nivel con el cuarto borra la variación muestral.

La distribución empírica

Una muestra observada induce la medida

\[ \widehat P_m=\frac1m\sum_{i=1}^m\delta_{(x_i,y_i)}. \]

Para cualquier función \(g\),

\[ \int g\,d\widehat P_m=\frac1m\sum_{i=1}^m g(x_i,y_i). \]

Un promedio calculado con datos es una esperanza bajo \(\widehat P_m\), no automáticamente bajo \(P\).

De promedio observado a esperanza

Cantidad poblacional

\[ \mu=\mathbb E_P[Y] \]

Es fija bajo el modelo, aunque sea desconocida.

Estimador

\[ \bar Y_m=\frac1m\sum_{i=1}^mY_i \]

Cambia si cambia la muestra.

Si las observaciones son iid y \(\operatorname{Var}(Y)<\infty\),

\[ \operatorname{Var}(\bar Y_m)=\frac{\operatorname{Var}(Y)}{m}. \]

Aprender significa aproximar un riesgo

Para una regla \(f\) y una pérdida \(\ell\):

Riesgo esperado

\[ R_P(f)=\mathbb E_P[\ell(Y,f(X))] \]

Describe desempeño bajo la población objetivo.

Riesgo empírico

\[ \widehat R_m(f)=\frac1m\sum_{i=1}^m\ell(y_i,f(x_i)) \]

Es calculable y depende de la muestra.

La distancia entre ambos no desaparece por escribir más decimales.

Objeto Ejemplo ¿Cómo se fija?

Aprender no es predecir

Ajuste

Usa observaciones con resultado conocido para determinar parámetros.

\[ (X_{\mathrm{train}},y_{\mathrm{train}})\longmapsto \widehat\theta \]

Uso

Mantiene \(\widehat\theta\) fijo y calcula una salida para nuevas entradas.

\[ x_{\mathrm{nuevo}}\longmapsto f_{\widehat\theta}(x_{\mathrm{nuevo}}) \]

¿Cómo se comporta el procedimiento en observaciones que no participaron en el ajuste?

Una regla para test

Si una elección cambió después de mirar test, test ya participó en el diseño.

Esto incluye escoger variables, transformaciones aprendidas, hiperparámetros, umbrales y hasta la semilla “que se ve mejor”.

Para esas elecciones necesitaremos un conjunto de validación o validación cruzada.

Filtración de información

Hay leakage cuando el procedimiento de entrenamiento accede a información que no estaría disponible al predecir.

Ejemplos:

  • normalizar con media y desviación calculadas sobre toda la tabla;
  • usar una variable registrada después del resultado;
  • imputar faltantes antes de separar;
  • seleccionar el modelo con la métrica de test.

Los datos futuros pueden ser distintos

\[ (X,Y)_{\mathrm{train}}\sim P_{\mathrm{train}},\qquad (X,Y)_{\mathrm{uso}}\sim P_{\mathrm{uso}} \]

Una evaluación histórica es informativa cuando ambas distribuciones son suficientemente comparables para el uso previsto.

Cambios de población, instrumento, política o tiempo pueden romper esa comparación.

Lo que una asociación no demuestra

Una tabla observacional puede mostrar que dos variables cambian juntas. Eso no identifica por sí solo el efecto de intervenir sobre una de ellas.

Si estrato_promedio y precio_reportado están asociados, ¿subir administrativamente el estrato produciría el cambio predicho por una recta?

Predicción y causalidad responden preguntas distintas.
