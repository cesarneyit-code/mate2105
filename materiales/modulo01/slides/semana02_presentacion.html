La columna de unos no viene de los datos: codifica el intercepto.

Intercepto

\[ \hat y_i = \theta_0 + \theta_1 x_{i1} + \cdots + \theta_{p-1}x_{i,p-1} \]

Matemáticamente

El intercepto permite una predicción base cuando las variables numéricas valen cero.

Computacionalmente

En implementación manual se agrega una columna de unos. En sklearn, fit_intercept=True la maneja internamente.

Hipótesis lineal

\[ h_\theta(x_i)=x_i^\top\theta \qquad\Longrightarrow\qquad \hat y=X\theta \]

fila de X→producto punto→predicción individual · matriz X→producto matricial→todas las predicciones

Qué significa equivocarse

\[ r(\theta)=X\theta-y \]

Residual

El residual no es un detalle técnico. Es la evidencia primaria de error del modelo.

  • signo: sobre/subestimación;
  • magnitud: tamaño del error;
  • patrón: diagnóstico de modelo.

Pérdida cuadrática

\[ J(\theta)=\frac{1}{2n}\|X\theta-y\|_2^2 \]

Cuadrado

Penaliza errores grandes.

Promedio

Hace comparable el error entre tamaños de muestra.

Factor 1/2

Limpia la derivada; no cambia el minimizador.

Derivación: escribir antes de derivar

\[ J(\theta)=\frac{1}{2n}(X\theta-y)^\top(X\theta-y) \]

\[ =\frac{1}{2n} \left(\theta^\top X^\top X\theta-2y^\top X\theta+y^\top y\right) \]

La derivación es una inspección de formas, no una memoria de fórmulas.

Derivación: gradiente

\[ J(\theta)=\frac{1}{2n} \left(\theta^\top X^\top X\theta-2y^\top X\theta+y^\top y\right) \]

\[ \nabla J(\theta)=\frac{1}{n}X^\top(X\theta-y) \]

Chequeo de forma: X.T @ (X @ theta - y) debe producir un vector de tamaño p.

Condición de óptimo

\[ \nabla J(\theta)=0 \]

\[ X^\top(X\theta-y)=0 \]

\[ X^\top X\theta=X^\top y \]

Ecuaciones normales

Resolver mínimos cuadrados equivale a resolver un sistema lineal.

\[ X^\top X\theta=X^\top y \]

El objeto central no es la fórmula cerrada. Es el sistema que se resuelve.

Advertencia profesional

Mala práctica

theta = np.linalg.inv(X.T @ X) @ X.T @ y

Mejor práctica

theta = np.linalg.solve(X.T @ X, X.T @ y)

Calcular la inversa explícita suele ser menos estable y menos claro que resolver el sistema.

¿Cuándo falla la ecuación normal?

Colinealidad

Columnas de X casi redundantes.

Escala

Variables en magnitudes muy distintas.

Dimensión

Muchísimas variables o matriz mal condicionada.

Cierre técnico

La regresión lineal múltiple no empieza con una línea: empieza con una matriz de diseño, una pérdida y un sistema lineal.
